# M5Module-Fan-Test
M5Stack Fan Module v1.1 (STM32F030) Test

## 概要
このプロジェクトは、M5Stack Fan Module v1.1 (STM32F030)をテストするためのPlatformIOプロジェクトです。M5Unifiedライブラリを使用して、ファンモジュールの制御を行います。

## 機能
- 現在の速度表示
- ファンモジュールのバージョン表示


## 必要なライブラリ
- M5Unified
- M5Module-Fan

## 接続方法
M5Stack本体のGroveポート（I2C）にFan Moduleを接続します。
